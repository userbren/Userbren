# Tremendos Autos — Plataforma de Compraventa de Vehículos

Aplicación web completa para la compra, venta y visualización 3D de vehículos.

## Tecnologías

- **Backend:** Node.js, Express
- **Frontend:** HTML, CSS, JavaScript
- **Base de datos:** SQLite
- **3D:** Visor 3D integrado
- **Despliegue:** Render

## Funcionalidades

- 🚗 Catálogo de vehículos con imágenes
- 🏠 Panel de administración para gestión de inventario
- 🖼️ Visor 3D de vehículos
- 🔍 Búsqueda y filtros
- 📱 Diseño responsive

## Instalación

```bash
cd tremendos_autos
npm install
node servidor.js
```
