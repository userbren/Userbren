# AI Editor — Editor de Código con IA

Editor de archivos con interfaz web que permite navegar, editar y gestionar archivos del sistema con asistencia de IA.

## Tecnologías

- **Backend:** Node.js, Express
- **Frontend:** React Native (Expo)
- **Librerías:** fs-extra, cors

## Funcionalidades

- Navegación de archivos del sistema
- Edición de código en tiempo real
- Integración con asistentes de IA
- Interfaz responsive
- Operaciones seguras con sandbox de archivos

## Instalación

```bash
cd server
npm install
node server.js
```
