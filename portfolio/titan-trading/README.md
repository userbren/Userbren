# Titan Trading — Sistema de Trading Automatizado

Sistema de trading algorítmico con análisis técnico, gestión de riesgo y señales de trading para múltiples activos.

## Tecnologías

- **Lenguaje:** Python
- **Librerías:** pandas, numpy, ta-lib
- **Interfaz:** GUI con tkinter
- **Datos:** CSV históricos de múltiples activos
- **Empaquetado:** PyInstaller

## Funcionalidades

- 📊 Análisis técnico con indicadores (RSI, MACD, medias móviles)
- ⚠️ Motor de riesgo con límites configurables
- 🔔 Generación de señales de compra/venta
- 📈 Carga de datos históricos (acciones, crypto, commodities)
- 🖥️ Interfaz gráfica para monitoreo
- 📝 Logging de operaciones

## Activos Soportados

- Acciones (AAPL, NVDA, TSLA)
- Cryptomonedas (BTC, ETH)
- Commodities (Oro, Petróleo, Plata)
