# GTA Demo — Prototipo Unity

Prototipo de mundo abierto en Unity con sistema de vehículos, personaje en tercera persona y físicas básicas.

## Tecnologías

- **Motor:** Unity (2022 LTS / 2023 LTS)
- **Lenguaje:** C#
- **Assets:** Modelos 3D básicos, físicas con WheelColliders

## Funcionalidades

- 🎮 Jugador con CharacterController (movimiento, saltos)
- 🚗 Vehículo con sistema de físicas (WheelColliders, Rigidbody)
- 📷 Cámara en tercera persona
- 🗺️ Escenario grande (500x500)
- ⚙️ Setup automático desde menú de Unity

## Cómo usar

1. Abrir proyecto en Unity Hub (2022 LTS+)
2. Ir a menú `Titan Team > GTA Demo > Setup Escena Completa`
3. ¡Listo para jugar!
