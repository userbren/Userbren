## Resumen Rápido

### Upwork
- **Título:** Desarrollador Python & JavaScript | Automatización, Bots y Desarrollo Web
- **Tarifa inicial:** 15-20 €/h
- **Estrategia:** Propuestas cortas (< 150 palabras), específicas, con pregunta al final

### Fiverr
- **4 Gigs:** Bots Python (30€) · Web/APIs (50€) · Automatización (25€) · Unity (40€)
- **3 paquetes por Gig:** Básico / Estándar / Premium
